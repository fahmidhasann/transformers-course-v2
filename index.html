Learning Mission (আমাদের উদ্দেশ্য)

Interactive Tracker

LLM (Large Language Model) এর "behind the scene" বা ভেতরের মেকানিজম স্পষ্টভাবে বোঝা। কোনো জটিল গাণিতিক ফর্মুলা বা কোডিং জটলা ছাড়া Conceptual Intuition নিজের আয়ত্তে আনা, যাতে পরবর্তীতে যেকোনো advanced LLM concept সহজে ধরতে পারেন।

Lesson 1

Big Picture Pipeline: একটি মানুষের লেখা বাক্য বা প্রম্পট কীভাবে ধাপে ধাপে প্রসেস হয়ে পরবর্তী শব্দ অনুমান করে, তার সামগ্রিক উচ্চস্তরের (High-Level) রূপরেখা বোঝা।

Lesson 2

Input Pipeline: মানুষের বাক্য কীভাবে Tokenization, Sub-words, Vectors (Embedding) ও Positional Encoding অতিক্রম করে গাণিতিক ফিঙ্গারপ্রিন্ট তৈরি করে তা শেখা।

Lesson 3

Core Attention Intuition: Self-Attention, Query (Q), Key (K), Value (V) এবং Softmax-এর ভেতরের মেকানিজম অনুধাবন করা।

Lesson 4-5

Multi-Head Stacking: একাধিক মনোযোগ হেড কেন লাগে, Dimension Splitting কী এবং Residual Skip Connections ও LayerNorm কীভাবে ট্রেনিং স্থিতিশীল করে তা বোঝা।

Lesson 6

LLM Gen Architecture: Modern Decoder-Only Transformers (যেমন GPT, LLaMA) কেন ব্যবহৃত হয় এবং Causal Masking কী তা জানা।
